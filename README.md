angular movie finder
====================

####Development
Make sure to run bower, then a local server.

####Routes
* Home: [/app/#/](http://nathan.codes/movie-finder/app/#/)
* Individual Movies: [/app/#/movie/:id](http://nathan.codes/movie-finder/app/#/movie/tt3268458)

####Features
* OMDB Api
* Autocomplete Search
* Foundation
